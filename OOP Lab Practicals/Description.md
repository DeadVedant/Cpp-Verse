# Object Oriented Programming (OOP) Lab Practicals in C++

This repository contains all the Object Oriented Programming (OOP) practical programs implemented in C++.  
The programs are created as part of academic laboratory work and demonstrate the core concepts of OOP using simple and understandable examples.

---

# Topics Covered

- Basic C++ Programs
- Classes and Objects
- Constructors
- Inline Functions
- Operator Overloading
- Type Conversion
- Inheritance
- Multilevel Inheritance
- Templates
- Exception Handling
- this Pointer
- Function Definitions Outside Class
- User Input and Output Handling

---

# Practicals Included

1. Calculator using arithmetic operators
2. Student result and class calculator
3. Class declaration and object creation
4. Rectangle class with constructor and inline function
5. Operator overloading using UserProfile class
6. Type conversion using Currency class
7. Multilevel inheritance using University, College, Department
8. this Pointer using Student class
9. Grocery Bill Calculator using Class Templates
10. ATM Withdrawal using Exception Handling

# How to Run

1. Clone the repository
2. Open the program in any C++ IDE or compiler
3. Compile the code
4. Run the executable file

Example:
```bash
g++ program.cpp -o program
./program
```
