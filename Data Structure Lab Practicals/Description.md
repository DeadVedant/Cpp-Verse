# Fundamentals of Data Structure (FDS) Lab Practicals in C++

This repository contains all Fundamentals of Data Structure (FDS) practical programs implemented in C++.  
The programs are designed to demonstrate basic and advanced data structure concepts using simple and understandable logic.

---

- Sorting Algorithms
- Linked Lists
- Circular Linked Lists
- Stack using Arrays
- Queue using Arrays
- Data Structure Operations
- Dynamic Memory Concepts
- FIFO and LIFO Techniques
